quotefeed – Quote providers¶

class pyalgotrade.quotefeed.base.QuoteFeed¶

Base class for pyalgotrade.quote.Quote providing feeds.

getDataSeries(instrument=None)¶

Returns the QuoteDataSeries for a given instrument.

Parameters:instrument (string.) – Instrument identifier. If None, the last registered instrument is returned.
Return type:pyalgotrade.dataseries.QuoteDataSeries.
getRegisteredInstruments()¶

Returns a list of registered intstrument names.

class pyalgotrade.quotefeed.csvfeed.YahooFeed¶

A pyalgotrade.quotefeed.base.QuoteFeed that loads quotes from a CSV file downloaded from Yahoo! Finance.

addQuotesFromCSV(instrument, path, timeZone=0)¶

Loads quotes for a given instrument from a CSV formatted file. The instrument gets registered in the quote feed.

Parameters:
  • instrument (string.) – Instrument identifier.
  • path (string.) – The path to the file.
  • timeZone (int.) – The timezone for quotes. 0 if quote dates are in UTC.
